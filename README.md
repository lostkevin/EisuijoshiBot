永水高校使用的QQ管理机器人,用于简化报名\注册等一系列日常工作
## Environment
+ Docker
+ Wine CoolQ
+ CQHttpApi
+ nonebot
    + 为支持重启功能,用modified_lib/下的plugin.py覆盖nonebot内的plugin.py
+ mySQL + python connector
    + pip install mysql
